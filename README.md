# アクキー・アクスタ カメラARプレビュー

ページを開くとウェブカメラを起動し、その映像の上に固定の `model.glb` を重ねて見た目や雰囲気を確認する静的アプリです。GitHub Pages のような静的ホスティングにそのまま置けます。

## できること

- ページアクセス時にカメラを自動起動
- カメラ映像の上に固定3Dモデルを重ねて表示
- モデルサイズ、上下位置、向き、自動回転速度をその場で調整
- 背面カメラと前面カメラの切り替え

## ローカル起動

静的ファイルなので、任意のローカルサーバーで配信すれば動きます。

```powershell
python -m http.server 4173
```

起動後に [http://localhost:4173](http://localhost:4173) を開いてください。

## GitHub Pages で使うとき

- `index.html`
- `model.glb`

この 2 ファイルをルートに置いたまま公開すれば動きます。GitHub Pages は HTTPS なのでカメラ起動条件も満たします。

## 補足

- この実装はカメラ映像への重ね表示です。床検出や空間固定は行いません
- 実寸の最終確認を厳密にしたい場合は、別途 WebXR / Scene Viewer / Quick Look などの端末AR実装が必要です
- ブラウザでカメラ許可が必要です
